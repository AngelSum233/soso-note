2. 闭包：一个函数（产生新的作用域）定义的局部变量、子函数的作用域在函数内，但是一旦离开了这个函数，局部变量就无法访问，所有通过返回子函数到一个变量f1的方法，让 f1指向堆中的函数作用域，这样可以使用局部变量i.

   eg:

   ```js
   function Foo() {
       var i = 0;
       return function() {
           console.log(i++);
       }
   }
   ```
