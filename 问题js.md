#### 运算符

& 按位与，|按位或，^按位异或，~取反，>>右移，<<左移

1的二进制表示为 0 0 0 0 0 0 1
3的二进制表示为 0 0 0 0 0 1 1
根据 ==& 的规则== 得到的结果为 0 0 0 0 0 0 0 1,十进制表示就是1

==|==跟 & 的区别在于 如果对应的位中任一个操作数为1 那么结果就是1

==^==运算符跟 | 类似，但有一点不同的是 如果两个操作位都为1的话，结果产生0

==~==是对位求反 1变0， 0变1

==<<== 向左移动 而 ==>>==向右移动，超过的位将丢失，而空出的位则补0

```` c
int n1=3;
int n2=4;
n1 ^= n2; //1
n2 ^= n1; //1
n1 ^= n2;

n1 0000 0011
n2 0000 0010
````
