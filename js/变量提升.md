### js中的变量提升

> 在js中不止有变量有提升，对于函数也有提升。==且函数提升会发生在变量提升之前==。

*****例：

````javascript
var a = 1;
function fn(){
    var a = 2;
    function a(){console.log(3);}
    return a;
    function a(){console.log(4);}
}
var b = fn();
console.log(b);

// 输出 2
````

解析：变量a会先被赋值为函数f a(){console.log(3);}，之后被重新赋值为函数f a(){console.log(4);}，最后被赋值为2。
